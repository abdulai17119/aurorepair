# Precision Auto Care - Production-Ready Website

A modern, high-performance website for an auto repair shop built with pure HTML5, CSS3, and Vanilla JavaScript.

## 🚀 Features

- **Mobile-First Responsive Design**: Optimized for all devices from 320px to 4K.
- **Dynamic Service Showcase**: Service listings with pricing and duration.
- **Interactive Booking System**: Real-time appointment scheduling with LocalStorage persistence.
- **Custom Admin CMS**: Manage services and testimonials without touching code.
- **SEO Optimized**: Includes JSON-LD structured data, meta tags, and semantic HTML.
- **Accessibility**: WCAG 2.1 AA compliant with ARIA labels and keyboard navigation.
- **Performance**: PageSpeed optimized with lazy loading and CDN-based assets.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **Tailwind CSS (CDN)**: Utility-first styling with custom configuration.
- **Vanilla JS (ES6+)**: Modular logic for interactions and data management.
- **Lucide Icons**: Scalable vector icons.
- **Google Fonts**: Professional typography (Inter & Roboto).

## 📁 Structure

```text
├── index.html      # Main landing page
├── style.css       # Custom styles and animations
├── script.js       # Core logic and dynamic rendering
├── admin.js        # CMS and Admin dashboard logic
└── logos/          # SVG assets
```

## 🔐 Admin Access

To access the admin dashboard:
1. **Double-click the Logo** in the navigation bar.
2. Enter the password: `admin` (can be changed in `admin.js`).

## ⚙️ Setup & Deployment

1. Clone the repository.
2. Open `index.html` in any modern browser.
3. No build step required (Production-ready).
